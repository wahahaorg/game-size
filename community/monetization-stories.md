# 赚钱与变现

> 没有理想主义。群里讨论最多的就是：这个站能赚多少钱、怎么赚更多钱。

## 收入规模一览

真实的收入数据：

```txt
个人站：
  "一个月AdSense差不多十几刀" — 图图啊（初期）
  "月入万刀" — 多位没露面的群友（成熟期）

比赛级数据：
  第一名 $999 奖励，月 UV 160K，PV 340K
  第二名 $888 奖励，月 UV 8.9K
  第三名 $666 奖励，月 UV 6.97K

群整体：
  部分群友部分网站总PV来到3280万/月
```

## 收入结构分析

### 第一阶段：AdSense 广告（入门）

这是最基础的变现方式。门槛最低，但天花板也最低。

```txt
"我目前收入$50+，一个月AdSense差不多十几刀的样子。"
— 图图啊
```

### 第二阶段：订阅 + 积分（进阶）

工具站建议加上订阅和积分包模式：

```txt
"积分包是个好东西，让那些大户能够根据自己需要买更多积分来使用。
建议大家都在常规订阅套餐之外增加积分包。
可以设计成订阅之后才能买积分包。
这样用户就会先订阅，而不是先买永久积分包。"
— 哥飞
```

### 第三阶段：卖站（高阶）

```txt
"有部分群友不太理解，既然是做出来的赚钱的产品，为什么要卖掉呢？
我给大家举个例子：
有个人叫小明，他之前都在上班，积蓄不多。
基于小明现有的资源，他可以做出一个月收入1K的网站。
如果小明还想放大收入，他的资源限制住他了。
于是小明只能守着目前的资源，每个月赚1K。
如果小明把月收入1K的站卖掉（假设出售价格是月收入的20-30倍），
他就有了一笔2-3万的资金。
用这笔钱，他可以做更强的外链、投放广告、
请人帮他做内容，做出月收入3K甚至更高的网站。"
— 哥飞
```

## 定价策略

```txt
"如果你的网站功能比较单一，给到用户的权益比较弱，
订阅套餐最低价格$9.9是没问题的。
但如果你想提升一下收入，那么可以试着把最低价从$9.9改成$19.9或者$29.9。
涨价会影响付费率，但其实并不会影响太大。
假设你从$9.9提价到了$29.9，有30%人选择了不付费，
那么你最终收入是翻倍的。
甚至50%人选择不付费，都是之前收入的1.5倍。"
— 哥飞
```

### 定价速查

| 调整 | 流失率 | 收入变化 |
|------|--------|---------|
| $9.9 → $19.9 | 30% 流失 | +40% |
| $9.9 → $19.9 | 50% 流失 | 持平 |
| $9.9 → $29.9 | 30% 流失 | +110% |
| $9.9 → $29.9 | 50% 流失 | +50% |

## RPM（每千次展示收入）

| 网站类型 | RPM 范围 |
|---------|---------|
| 工具站 | ~$5 |
| 游戏攻略站（内容站） | $50 ~ $100 |
| 金融/商务类内容 | 更高 |

```txt
"单价主要看内容和国家，美国是最高的。
内容跟钱相关就会比较高。
有些关键词没什么搜索量，做到首页流量也不高，
但是单价高，也挺不错的。"
— 尼安
```

## 流量渠道分布

```txt
"目前流量都是bing带来的。"
— fiona（做游戏资讯站的群友）
```

不是所有流量都来自 Google。Bing 在某些领域能贡献大量流量，而且 Bing 的竞争远小于 Google。

## 收入进阶路径

```txt
第一步：AdSense（几百~几千刀/月）
   需要大量流量，单价低
第二步：订阅 + 积分（几千~几万刀/月）
   需要产品价值和用户粘性
第三步：卖站回笼资金（一次性收入）
   用于做更大的站
第四步：多站运营（总投资回报）
   分散风险，放大收入
```
