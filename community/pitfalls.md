# 避坑指南

> 群里真实发生过的翻车案例。踩过的坑不用再踩一遍。

## 技术类

### 坑 1：API Key 埋在前端代码里

```txt
"AI写完了，没和AI聊安全问题。
前端js代码里直接写了 API Key。"
— 星辰
```

**教训：** 每次 AI 生成代码后，检查 API Key、密钥、Token 是否暴露在前端。

### 坑 2：重定向配置错误导致排名消失

```txt
"本来追了个新词谷歌排名前3了，然后我改到了网页自动重定向了，
排名掉到40了，后面发现了彻底回退版本回去了。"
— Zaro

"我也不知道怎么改的，我是技术小白，
然后就让trae直接改了，估计是改乱了，
然后发版本就发现网页整个进不去了，
然后就继续让AI改，我自己不会看，
估计就越改越乱了。"
— Zaro
```

**教训：** 不懂技术时不要随意改服务器配置。改动前先备份，改完先在小范围验证。

### 坑 3：VPS 出口 IP 不干净

```txt
"自建节点如果想要干净的出口IP，也要提前查一下。
一些小的VPS供应商，因为卖的实例比较便宜，会有人滥用。
买到的出口IP可能是复用的，还是不能保证干净。"
— LIN_HAN
```

## SEO 类

### 坑 4：改版幅度过大

```txt
"修改了标题描述，还大改了布局，差不多20天左右才恢复。
Bing直接清零掉了。"
— 阿飞

"不建议新手去改，因为不可控，尺度把握不好就容易gg。"
— 阿飞
```

详见 [改版代价与维护](./update-risks) 章节。

### 坑 5：Canonical 标签忘记改

```txt
"可以检查一下是不是页面模板里面的Canonical参数忘记改了。"
— NightWatch
```

换模板或改 URL 结构时，Canonical 标签是容易被忽略但影响很大的细节。

### 坑 6：重复内容导致关联封号

```txt
"多个网站内容都差不多，被关联，然后谷歌买量账户被封了。"
— 兔猫肉未滋油
```

## AdSense 类

### 坑 7：太新的网站申请 AdSense

```txt
"太新的网站没流量，审核比较难过。"
— 尼安
```

刚上线的网站就去申请 AdSense 基本都会被拒。建议等有内容积累和稳定流量后再说。

### 坑 8：不同审核员标准不一致

```txt
"有时看审核员的，我通过20多个网站，最多的一个申请了7次，
什么都没改，通过了。"
— 尼安
```

被拒不一定是内容问题。可以隔段时间再试。

## 建站策略类

### 坑 9：做跟主业不相关的工具引流

```txt
"这个站的页面SEO细节还是做得不错的，是懂行的人做的。
但是选择关键词却选错了。
这些工具带来的流量，基本不会转化为他主业的付费用户。"
— 哥飞
```

流量不能变现就是无效流量。做工具站时确保工具的用户跟你网站的变现目标一致。

### 坑 10：前期没考虑订阅模式

```txt
"我是做了几年的内容站，被AI影响到了才来做AI工具站。
目前每天1000多访客，但是没订阅，前期没想过订阅，
单靠广告不行。"
— 尼安
```

工具站**必须设计订阅模式**，单靠广告很难支撑。

### 坑 11：免费试用被薅羊毛

```txt
"我的图片工具站设置的是登录后可以免费生成一张，
但是发现有个人换不同的邮箱（域名都一样，用户名随机）来薅羊毛，
这种怎么处理比较好？"
— Lok
```

方案：限制同一 IP 的免费次数、使用设备指纹校验、增加邮箱验证门槛。

## 工具类

### 坑 12：长期依赖某个免费工具

```txt
"Folo解散了。核心原因就是早期没想好商业化，
于是后面就无以为继了。"
— 哥飞
```

不要依赖不收费的工具或服务来做核心业务，它们随时可能关闭。

### 坑 13：AI 工具不稳定

```txt
"ChatGPT暴毙了，天天要出点幺蛾子。"
— Excited!
```

把 AI 工具当辅助，不要当成唯一的工作流。
