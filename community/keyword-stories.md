# 关键词实战

> "长尾词拿到了53%的点击，其他网站几乎没有对这个词做关键词密度。"

## 大词 vs 长尾词的策略

### 大词的问题

```txt
"发现是一个很小的长尾词，拿到了53%的点击，
其他网站几乎没有对这个词做关键词密度。
又学到了，一直在怼大词，
没有好好研究一下长尾词。"
— 陈余忻
```

**教训：** 所有人都盯着大词，竞争极其激烈。很多小词/长尾词因为竞争对手干脆没做优化，你稍微做一下就能拿到大部分点击。

### 农村包围城市策略

```txt
"先调研某大类里比较细分的一个词，这个词难度比较低。
功能放到网站的内页里面。
先想用这个词作为切入点，把小词做起来，
然后慢慢一步步也就能打通大类。"
— 西西弗斯
```

**关键词域名** 也是群里常用策略：

```txt
"做好了飞哥说的关键词域名和精品页SEO，
应该至少能排进前两页，竞争大的除外。"
— Kai
```

## 关键词密度的真实作用

```txt
"关键词密度这个指标，只是用来确认大家的一个网页
真的是围绕着一个关键词主题去写的。"
— 哥飞
```

不是排名因素，是**相关性检查工具**。密度高不等于排名好，但密度太低说明页面可能没有聚焦主题。

## 关键词的搜索意图

### 工具类关键词

```txt
"群友做了一个AI故事生成器页面：
https://www.appointo.me/tools/ai-story-generator
这个站的页面SEO细节还是做得不错的，是懂行的人做的。
但是选择关键词却选错了，不知道为什么要选这些关键词。
这些工具带来的流量，基本不会转化为他主业的付费用户。"
— 哥飞
```

**教训：** 关键词不仅要看搜索量，还要看**搜索意图是否能转化为你的变现目标**。如果你的网站是做 SaaS 的，吸引来用一用就走的故事生成器用户毫无意义。

### 单价关键词

```txt
"有些关键词没什么搜索量，做到首页流量也不高，
但是单价高，也挺不错的。"
— 尼安
```

推荐用 Ahrefs 免费工具或 Google Keyword Planner 查关键词的估算出价：

```txt
"Keywords Explorer关键词探索器，这个是Ahrefs上付费的功能。"
— 张军-Kimi
```

## 关键词的时效性

游戏关键词有强烈的时效性：

- 新游戏/新版本上线 → 需求爆发 → 竞争极少 → 最佳窗口
- 活动期间 → 短期需求高峰
- 版本末期 → 需求下降 → 内容贬值

**做游戏站关键词的关键：** 关注新版本公告、关注活动日历。在需求起来之前先把内容做好。
